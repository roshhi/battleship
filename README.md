# Raindrop

Raindrop is a fully responsive weather application that provides real-time weather updates, weekly forecasts, and key highlights such as humidity, wind speed, air quality, UV index, sunrise/sunset, and visibility.

## Features

- Auto-detect user location via BigDataCloud API
- Real-time weather data from OpenWeatherMap & Open-Meteo APIs
- 7-day weekly forecast
- UV Index & Sunrise/Sunset timings (TimezoneDB API)
- Wind speed & direction
- Humidity levels
- Visibility status
- Air Quality index
- 100% Responsive – works on desktop, tablet, and mobile
- Multiple Theme Support – UI dynamically adapts based on current weather conditions (rainy, snowy, sunny, cloudy, etc.)
